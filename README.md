# Sparkle

A free, no-ads nail art app for kids. Pick a hand, paint nails, add sparkles and emoji stickers — pure creative play, nothing else.

## Features

- Cartoon hand with adjustable skin tone
- Full color palette for painting nails
- Sparkle effects and emoji sticker overlays
- Toggle between left and right hand
- Mobile and tablet friendly

## Tech

React · TypeScript
